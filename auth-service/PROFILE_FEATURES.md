# 👤 Личный кабинет пользователя

## 🚀 Новые возможности

Мы успешно добавили полнофункциональный личный кабинет в auth-service со следующими возможностями:

### ✨ Основные функции

#### 1. **Управление профилем**
- ✅ Редактирование личных данных (имя, email, телефон, должность, отдел)
- ✅ Загрузка и обновление аватарки (поддержка JPG, PNG, GIF, до 99MB)
- ✅ Смена пароля с проверкой текущего
- ✅ Мгновенное обновление данных без перезагрузки страницы

#### 2. **Просмотр ролей**
- ✅ Отображение всех ролей пользователя по сервисам
- ✅ Группировка ролей по сервисам с отображением названий
- ✅ Визуальное представление ролей в виде бейджей

#### 3. **Управление документами**
- ✅ Загрузка документов (TXT, PDF, DOC, DOCX, до 10MB)
- ✅ Drag & Drop интерфейс для удобной загрузки
- ✅ Просмотр списка загруженных документов
- ✅ Скачивание документов
- ✅ Удаление документов с подтверждением

### 🔧 Технические изменения

#### **Backend (Go)**
- 📝 Расширение модели User новыми полями:
  - `Phone` - телефон
  - `Position` - должность  
  - `Department` - отдел
  - `AvatarPath` - путь к аватарке
  - `Documents` - массив документов
  - `CreatedAt/UpdatedAt` - временные метки

- 🛠 Новые функции в models/user.go:
  - `UpdateUserProfile()` - обновление профиля
  - `UpdateUserAvatar()` - обновление аватарки
  - `AddUserDocument()` - добавление документа
  - `RemoveUserDocument()` - удаление документа
  - `ChangeUserPassword()` - смена пароля

- 🌐 Новые роуты в routes/auth.go:
  - `GET /auth/profile` - страница профиля
  - `POST /auth/profile/update` - обновление профиля
  - `POST /auth/profile/avatar` - загрузка аватарки
  - `POST /auth/profile/password` - смена пароля
  - `POST /auth/profile/document` - загрузка документа
  - `POST /auth/profile/document/delete` - удаление документа
  - `GET /auth/profile/document/:id` - скачивание документа

#### **Frontend**
- 🎯 Новый шаблон `profile.html` с современным дизайном
- 📱 JavaScript для асинхронных операций
- 🎨 CSS стили для всех компонентов
- 🔄 AJAX запросы без перезагрузки страницы

#### **Файловая система**
- 📁 Созданы директории для хранения файлов:
  - `/data/avatars/` - аватарки пользователей
  - `/data/documents/` - документы пользователей
- 🔗 Настроена раздача статических файлов через Gin

### 🎯 Доступ к личному кабинету

1. **Через главное меню** - карточка "Личный кабинет"

3. **Прямая ссылка** - `/auth/profile`
